Game of Life (ASCII, C)

===========================
Описание
---------------------------
Это интерактивная реализация "Игры Жизни" Джона Конвея на языке C с использованием только символов ASCII для вывода в терминал. Размер игрового поля — 80x25 ячеек, поле замкнуто по краям.

---------------------------
Как собрать игру
---------------------------
1. Убедитесь, что у Вас установлен компилятор gcc.
2. Перейдите в корневую папку проекта.

3. Скомпилируйте программу командой:
   gcc -o game_of_life src/game_of_life.c -std=c99

---------------------------
Как запустить игру
---------------------------
1. Убедитесь, что в папке datasets/ есть файлы 1.txt, 2.txt, 3.txt, 4.txt, 5.txt с начальными состояниями.
2. Запустите игру командой:
   ./game_of_life

3. При запуске программа попросит ввести номер датасета (от 1 до 5). Введите нужный номер и нажмите Enter.

---------------------------
Управление во время игры
---------------------------
A — увеличить скорость (ускорить игру)
Z — уменьшить скорость (замедлить игру)
Пробел — завершить игру

---------------------------
Примечания
---------------------------
- Чтобы добавить свои начальные состояния, создайте дополнительные файлы .txt в папке datasets/ формата, аналогичного существующим.
- Для очистки экрана используется ANSI-escape последовательность, поэтому корректная работа ожидается в большинстве современных терминалов.
- Игра не использует system() и другие запрещённые системные вызовы.
- Для запуска на Windows используйте терминал, поддерживающий ANSI-escape коды (например, Windows Terminal).

---------------------------
Структура проекта
---------------------------
include/game_of_life.h       — заголовочный файл
src/game_of_life.c           — исходный код игры
datasets/1.txt...5.txt       — файлы с начальными состояниями
